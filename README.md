# RetroSnake

### Um jogo clássico de Snake feito em C++ utilizando a biblioteca Raylib.

---

## Tutorial/Controle

- **M**: Tirar a música
- **Espaço**: Pausar
- **ESC**: Sair
- **Setas**: Movimentar
